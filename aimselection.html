Aim of Selection Sort


•To sort an array of elements into ascending or descending order.


•To repeatedly find the minimum (or maximum) element from the unsorted portion of the array and place it at the beginning (or end) of the sorted portion.


•To maintain two subarrays within the original array: a sorted subarray and an unsorted subarray.


•To start with a sorted subarray of zero elements and an unsorted subarray of all elements.


•To expand the sorted subarray by one element with each iteration by finding the minimum (or maximum) element from the unsorted subarray and swapping it with the first element of the unsorted subarray.

•To repeat the process until the entire array is sorted.
