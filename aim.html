Aim of Bubble Sort


➤ an unsorted array of numbers, generate a sorted array of numbers by applying Bubble Sort.


➤ Optimise the Bubble Sort algorithm to achieve better performance.


➤ Demonstrate knowledge of time complexity of Bubble Sort by counting the number of

operations involved in each iteration


➤ To repeat the comparison and swap process until the entire array is sorted.


➤ To have a time complexity of O(n^2), which is less efficient than other sorting algorithms

for larger datasets, but can still be effective for small datasets.
